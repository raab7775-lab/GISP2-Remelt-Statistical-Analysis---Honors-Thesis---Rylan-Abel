# GISP2-Remelt-Statistical-Analysis---Honors-Thesis---Rylan-Abel
This repository contains the code used in Rylan Abel's honors thesis "Data Analysis and Comparison of the New GISP2 Ice Core Data: Continuous High-Resolution Water Isotope Data Vs Discrete 1997 and 2004 Collection". It includes a copy of the code used for the interval within the thesis, the data used (edited for direct application into code and unedited with metadata), and a template for whole-core analysis.
